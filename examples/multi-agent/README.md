# Multi-Agent Examples

## Usage

```sh
# Validate
pnpm cli -- multi-agent validate examples/multi-agent/safe-planner-executor-workflow.json

# Run Safe Workflow
pnpm cli -- multi-agent run-demo examples/multi-agent/safe-planner-executor-workflow.json

# Run Blocked Escalation
pnpm cli -- multi-agent run-demo examples/multi-agent/blocked-privilege-escalation-workflow.json
```
